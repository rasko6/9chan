# 9chan

Анонимный имиджборд в немецком стиле

## Доски

- **/b/** — Разное
- **/g/** — Технологии и программирование
- **/a/** — Аниме и манга
- **/pol/** — Политика
- **/d/** — Изображения
- **/k/** — Оружие

## Функции

- Создание тредов с изображениями
- Ответы в тредах
- Капча при создании треда и ответов
- Синхронизация через GitHub Gist
- Адаптивный дизайн
- Немецкая цветовая гамма


## Технологии

- HTML5
- CSS3
- JavaScript
- GitHub Gist API

# 9chan 

Анонимный имиджборд в немецком стиле

<img width="100" height="100" alt="image" src="https://github.com/user-attachments/assets/23ef861f-e16c-4dd0-abe9-e4f6990b179f" />

## Доски

- **/b/** — Разное
- **/g/** — Технологии и программирование
- **/a/** — Аниме и манга
- **/pol/** — Политика
- **/d/** — Изображения
- **/k/** — Оружие
- 
## Функции

- Создание тредов с изображениями
- Ответы в тредах
- Капча при создании треда и ответов
- Адаптивный дизайн
- Немецкая цветовая гамма
